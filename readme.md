3. 每日http指南:
   	1. 媒体类型MIME，它是一种文本标记，表示一种主要的对象类型和一个特定的子类型，中间由一条斜杠来分隔。
    	2. URI分为URL和URN，主要使用URL，URN还处于实验阶段
    	3. URL分为三部分：
        	1. 第一部分为方案，说明了访问资源所使用的协议类型。HTTP协议
         	2. 第二部分给出了服务器的因特网地址
         	3. 其余部分指定了web服务器上的某个资源
